# User Story - Visualização de Gastos Pessoais

## Descrição

Funcionalidade voltada para usuários que desejam acompanhar despesas mensais e identificar gastos desnecessários através de tabelas e informações organizadas.

---

## User Story

Como Fernanda,  
quero visualizar meus gastos mensais em uma tabela organizada no aplicativo,  
para identificar despesas desnecessárias e melhorar meu controle financeiro.

---

## Objetivo

Facilitar a visualização dos gastos pessoais através de informações claras e organizadas.

---

## Benefícios Esperados

- Melhor controle dos gastos
- Identificação de despesas desnecessárias
- Organização financeira pessoal
- Facilidade de visualização das despesas
