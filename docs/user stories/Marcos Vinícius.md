# User Story - Controle de Patrimônios

## Descrição

Funcionalidade voltada para usuários que desejam acompanhar patrimônios e visualizar o valor total dos bens de forma organizada.

---

## User Story

Como Marcos,  
quero visualizar meus patrimônios e o valor total dos meus bens no aplicativo,  
para acompanhar minha situação financeira de maneira mais organizada.

---

## Objetivo

Permitir o acompanhamento centralizado e organizado dos patrimônios do usuário.

---

## Benefícios Esperados

- Melhor organização patrimonial
- Facilidade de acesso às informações
- Visualização clara do valor total dos bens
- Centralização das informações financeiras
