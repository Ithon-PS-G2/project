# Wireframe do Sistema

## O que é um Wireframe?

Um **wireframe** é uma representação visual simplificada da interface de um sistema ou aplicação. Ele funciona como um esboço estrutural das telas, mostrando a disposição dos elementos, menus, botões, formulários e conteúdos antes do desenvolvimento final.

Os wireframes são utilizados para:
- Planejar a organização das telas;
- Melhorar a experiência do usuário;
- Validar ideias de interface;
- Facilitar a comunicação entre equipe e clientes;
- Servir como base para o desenvolvimento do sistema.

No projeto do **Sistema Simplificado de Gestão Familiar**, os wireframes foram desenvolvidos utilizando a plataforma Figma para auxiliar na prototipação das telas e definição da interface do usuário.
